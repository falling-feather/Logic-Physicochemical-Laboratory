## 一、内容规划（对齐人教版 2019 新课标）

### 规划原则

1. **覆盖度**：优先覆盖必修教材核心知识点，选择性必修渐进补充
2. **交互深度**：每个实验必须包含**原理层**（展示底层机制）+ **操作层**（用户可调控参数）+ **总结层**（关键公式/结论）
3. **教育意义**：拒绝蜻蜓点水式动画，必须让用户理解"为什么"

#### 化学反应模块优化方案（重点）

**当前问题**：
- 原子仅表现为圆形粒子，缺乏化学键可视化
- 反应过程只有"聚集→抖动→分散"，未展示键的断裂与形成
- 没有活化能概念、没有反应历程
- 缺乏教育深度

**优化目标**：
1. **分子建模**：反应物以分子结构（带化学键）呈现，而非散乱原子
2. **键断裂动画**：化学键断开时有视觉效果（虚线→断裂→消失）
3. **键形成动画**：新键形成时原子逐渐靠近并连接
4. **能量图**：显示反应进程-能量曲线（含活化能峰值）
5. **氢键/离子键**：不同键型使用不同视觉样式（虚线、实线、粗细）

## 三、目录跳转优化

### 当前跳转路径
```
首页卫星 → 学科页面 Hero → 模块选择器画廊 → 实验内容
导航栏    →                   ↗
实验卡片  →    ↗
```

### 优化方案

1. **面包屑导航**：实验页面顶部显示 `首页 > 化学 > 分子结构`
2. **学科内目录**：侧边栏显示当前学科所有实验，可快速跳转
3. **桌面端快捷键**：`Ctrl+1~5` 快速切换学科
4. **搜索功能**：全局搜索实验名/关键词
5. **跨学科推荐**：实验底部推荐关联实验（如 DNA → 遗传学）
