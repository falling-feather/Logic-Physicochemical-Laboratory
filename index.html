波动原理

波的叠加、驻波与多普勒效应

1

波的叠加 — 多列波在同一介质中传播时，合位移等于各波位移的矢量和。拍频波产生包络线。

驻波 — 两列同频反向波叠加， y = 2A sin(kx) cos(ωt)，波节与波腹位置固定。

流体力学原理

理想不可压缩流体的基本定律

1

势流叠加 — 理想流体中，均匀流、点源/汇、点涡等基本流动可线性叠加，构造复杂流场。

2

圆柱绕流 — 均匀流 + 偶极子 = 无环量绕流；叠加自由涡 Γ 产生 Magnus 效应，升力 FL = ρUΓ。

3

伯努利方程 — P + ½ρv² = const，管道收窄处流速增大、压强降低（Venturi 效应）。

4

连续性方程 — A₁v₁ = A₂v₂，不可压缩流体的质量守恒要求截面积与流速成反比。

光学原理

几何光学与波动光学的基本定律

1

薄透镜公式 — 1/f = 1/do + 1/di，焦距不变时物距决定成像特性（放大/缩小、实像/虚像）。

2

双缝干涉 — 两缝发出的相干球面波叠加，条纹间距 Δy = λL/d，波长越长或缝距越小条纹越宽。

3

Snell 定律 — n₁sinθ₁ = n₂sinθ₂，光从光密到光疏介质时，入射角超过临界角 θc 发生全反射。

4

棱镜色散 — 不同波长折射率不同（Cauchy 方程 n(λ) = A + B/λ²），白光经三棱镜分解为连续光谱。

5

衍射光栅 — N 缝等间距干涉，主极大条件 d sinθ = mλ，缝数越多峰越尖锐，白光在各级次展为彩色光谱。

6

偏振与 Malus 定律 — 线偏振光通过检偶器 I = I₀cos²θ，两片正交偏振片之间插入 45° 中间片反而透光（偏振悖论）。

桶排序 — 算法原理

桶排序（Bucket Sort）是一种分配排序算法：

  1. 1

    分配阶段 — 将待排序数组中的元素根据其值的大小，分配到不同的"桶"中。

  2. 2

    排序阶段 — 对每个非空的桶内的元素进行排序。

  3. 3

    收集阶段 — 按照桶的顺序，依次将每个桶中的元素取出，合并成最终的排序结果。

搜索算法

二分查找、DFS/BFS 等搜索过程可视化

搜索算法对比 — 四种经典方法

在有序数组中查找目标值的四种策略，复杂度从 O(n) 到 O(log log n)：

  1. L

    线性搜索 O(n) — 逐个比较，最简单但最慢。不要求有序。

  2. B

    二分查找 O(log n) — 每次排除一半，20 个元素最多 5 步。

  3. I

    插值查找 O(log log n) — 根据值的分布估算位置，均匀数据极快。

  4. J

    跳跃查找 O(√n) — 先以 √n 步长跳跃定位区间，再线性扫描。

二叉树遍历

前序、中序、后序与层序遍历动画

二叉树遍历 — 四种经典方式

访问二叉树所有节点的四种策略，区别在于根节点的访问时机：

  1. 前

    前序 (NLR) — 根→左→右。用于复制树、前缀表达式。根在第一位。

  2. 中

    中序 (LNR) — 左→根→右。BST 的中序输出即为有序序列！

  3. 后

    后序 (LRN) — 左→右→根。适合删除树、后缀表达式求值。

  4. 层

    层序 (BFS) — 用队列逐层访问。适合求最大宽度、层序打印。

哈希表查找

冲突处理与负载因子对性能的影响

哈希表查找 — O(1) 的秘密

哈希表通过哈希函数将键映射到槽位，实现近乎常数时间的查找：

  1. H

    哈希函数 — hash(key) = key % tableSize，将任意键映射到有限槽位。

  2. C

    冲突处理（链地址法） — 同一槽位的多个元素形成链表，逐个比较。

  3. α

    负载因子 α = n/m — 元素数/槽数。α 越大冲突越多，性能从 O(1) 退化到 O(n)。

图算法 — 最短路径 & 最小生成树

两种经典的贪心图算法，用于加权无向图：

  1. D

    Dijkstra 最短路径 — 从起点出发，每次选距离最小的未确认节点，松弛其邻居的距离估计。保证找到从起点到所有节点的最短路径。

  2. P

    Prim 最小生成树 — 从任意节点开始，每次选一条连接已访问和未访问节点的最小权边，加入生成树。保证总权值最小。

  3. !

    对比关键 — Dijkstra 关注到源的累计距离，Prim 关注单条边的权值。点击节点可更换起点。
