## 四、目录跳转优化

### 当前跳转路径

```
首页卫星 → 学科页面 Hero → 模块选择器画廊 → 实验内容
导航栏    →                   ↗
实验卡片  →    ↗
Skip-nav  →    直接跳转主内容区
```

### 优化方案

1. **面包屑导航**：实验页面顶部显示 `首页 > 化学 > 分子结构`
2. **学科内目录**：侧边栏显示当前学科所有实验，可快速跳转
3. **桌面端快捷键**：`Ctrl+1~5` 快速切换学科
4. **搜索功能**：全局搜索实验名/关键词
5. **跨学科推荐**：实验底部推荐关联实验（如 DNA → 遗传学 → 基因突变）
