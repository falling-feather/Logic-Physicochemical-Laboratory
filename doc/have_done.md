## 一、内容规划完成总结（对齐人教版 2019 新课标）

### 规划原则

1. **覆盖度**：优先覆盖必修教材核心知识点，选择性必修渐进补充
2. **交互深度**：每个实验包含**原理层**（展示底层机制）+ **操作层**（用户可调控参数）+ **总结层**（关键公式/结论）
3. **教育意义**：拒绝蜻蜓点水式动画，必须让用户理解"为什么"

## 十、2026-04-17 生物模块修复 + 缓存优化（v4.0.1）

### 10.1 生物模块语法修复

| 文件                   | 问题                                                                                                  | 修复                                                                                                |
| ---------------------- | ----------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| `cell-structure.js`  | `_detGeneric()` 与 `_injectInfoPanel()` 方法之间缺少逗号，导致 SyntaxError: Unexpected identifier | 补充逗号分隔符                                                                                      |
| `gene-mutation.js`   | `updInfo()` 中模板字面量闭合使用单引号而非反引号，导致 SyntaxError: Unexpected end of input         | 修正为反引号闭合                                                                                    |
| `module-selector.js` | 生物模块 init 函数映射不正确（未调用实际 init 函数）                                                  | 修正为 `initCellStructure()` / `initDNAHelix()` / `initPhotosynthesis()` / `initGenetics()` |

- `window.CellStructure = CellStructure`
- `window.initCellStructure = initCellStructure`
- `window.initGeneMutation = initGeneMutation`

### 10.2 回访用户缓存优化（两层架构）

#### 第一层：localStorage 回访检测

- `index.html` 内联脚本在解析早期读取 `englab-cache-meta` key
- 存储结构：`{ visitCount, lastVisit, hasSeenSplash }`
- 回访用户标识：`html.return-visit` CSS 类 + `window.__englabCache.returning = true`
- 效果：
  - 加载屏过渡时间 0.5s → 0.26s，loader 尺寸缩小
  - `main.js` 轮询间隔 90ms → 60ms，fallback 超时 1800ms → 900ms
  - `home.js` 减少星星数量，延迟加载粒子网络/流星等非关键动画
  - 新增重复绑定防护（parallax / eye-tracking / shooting-stars）

#### 第二层：Service Worker 离线缓存

- 新增 `sw.js`：缓存名 `englab-static-v20260416b`
- 安装阶段预缓存 APP_SHELL（index.html、核心 JS/CSS、首页资源）
- 导航请求：network-first + cache fallback
- 静态资源：stale-while-revalidate（.js/.css/.png/.jpg/.svg/.woff2）
- `main.js` 通过 `requestIdleCallback` 延迟注册 SW，不阻塞首屏
- 版本管理：更新代码后需同步更新 `sw.js` 中 `CACHE_NAME` 和 `index.html` 中 `?v=` 参数

### 10.3 验证结果

- 首次访问加载屏消失时间：~1243ms
- 二次访问（SW 缓存命中）：~678ms（提速约 45%）
- 生物模块 `initCellStructure` / `initGeneMutation` 均恢复为 function 类型
- Cell Structure 画布正常渲染动画，Gene Mutation 突变触发正常工作
